Chào mừng đến với bài học CSS

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ dùng để tạo kiểu dáng cho trang web. Nó giúp thay đổi màu sắc, định dạng chữ, căn chỉnh bố cục, v.v.

Những lệnh CSS thường gặp

  • Background: background-color, background-image
  • Font: font-family, font-size, font-weight
  • Color: Định dạng màu sắc của chữ
  • Text: text-align, line-height
